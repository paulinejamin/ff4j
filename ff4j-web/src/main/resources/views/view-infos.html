Donation

The FF4J team has been working very hard to provide a powerful Feature Toggle solution which is stable, highly tested, as well as easy to configure and use. We always make every effort to respond to all support and request queries in a timely manner and work with users to help resolve their problems quickly. All of this requires a surprising amount of time.

As open source application, it is free and will always remain free. However, if you enjoy using ff4j or find that it is improving the value of a product or service that your organization provides, we urge you to consider donating.
